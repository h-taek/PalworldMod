# MinimapWidget — 미니맵 모드

팰월드용 자체제작 미니맵. 정적 전체지도를 원형으로 크롭해 화면 구석에 띄우고, 크기·줌·위치를
인게임 ModConfigMenu에서 실시간 조정한다. 던전·보스타워 같은 지하 인스턴스에 들어가면
자동으로 숨는다.

## 기능

- 원형 미니맵 (전체지도 크롭 + 링 프레임)
- 크기·줌·위치를 인게임에서 실시간 조정 (ModConfigMenu 패널)
- 주요 지점 마커 — 워프·타워·던전·여신상·기지·관측탑·필드보스·현상수배, 나침반 POI,
  플레이어 방향 화살표
- 던전·보스타워 진입 시 자동 숨김

## 선행 요구 (필수)

- **UE4SS-Palworld-macOS 로더** — 이 모드 구동의 전제
- **ModConfigMenu** — 크기/줌/위치 설정 UI (별도 설치)
- **BPModLoaderMod** — 인게임 Mod Config 메뉴 스폰

## 설치

**권장 — 모드 매니저 앱**
릴리즈 zip을 PalworldModManager로 가져오면 설치·활성·자동업데이트가 처리된다.
설정 저장에 필요한 처리도 매니저가 대신 해준다.

**수동 설치(드롭인)**
zip 안의 `Pal/` 폴더를 게임 프로젝트 폴더(`Palworld.app/Contents/UE/Pal/`)에 병합한다.
(`manifest.json`·`README.txt`는 게임에 넣지 않는다 — 매니저/안내용)

## 주의

- **macOS(Apple Silicon)에서 테스트됨** — 윈도우에서도 동작할 수 있으나 검증되지 않았다.
- 설정(크기/줌/위치) 저장은 macOS 샌드박스 제약 때문에 별도 우회 처리가 필요하다 —
  매니저 앱이 자동으로 처리하며, 순수 수동 설치 시에는 저장이 반영되지 않을 수 있다.
