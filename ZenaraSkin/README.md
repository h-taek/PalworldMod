# ZenaraSkin — 제나라 외형 스킨

팰월드 플레이어의 외형을 게임 내 최종 탑 보스 **제나라**(코드명 `WorldTreeBoss`)로 바꾸는
스킨 모드다. 색·텍스처를 게임 원본 자산에서 그대로 빌려 쓰기 때문에, 추가 텍스처나 외부
"머리 숨김" 모드 없이 **혼자서 완결되게** 동작한다.

## 기능

- 플레이어 외형을 제나라로 교체 — 얼굴·몸·의상·머리카락 포함
- 게임 원본 재질/텍스처를 그대로 참조 → 정확한 색감, 별도 텍스처 파일 불필요
- 바닐라 머리·머리카락을 자동으로 숨김
- 기본 의상 슬롯 전체에 적용 → 어떤 옷을 입어도 제나라 외형 유지

## 설치

**권장 — 모드 매니저 앱**
릴리즈 zip을 PalworldModManager로 가져오면 설치·활성·자동업데이트가 처리된다.

**수동 설치(드롭인)**
zip 안의 `Pal/` 폴더를 게임 프로젝트 폴더(`Palworld.app/Contents/UE/Pal/`)에 병합한다.
(`manifest.json`·`README.txt`는 게임에 넣지 않는다 — 매니저/안내용)

## 주의 · 호환

- **다른 전신 스킨과 동시 사용 불가** — 같은 의상 슬롯을 덮어쓴다. 한 번에 하나만 활성.
- **macOS(Apple Silicon), CrossOver windows 에뮬에서 테스트됨**
- **게임 버전 종속** — 대규모 업데이트로 원본 보스 자산 경로가 바뀌면 외형이 깨질 수 있다.

## 동작 방식 (요약)

게임의 제나라 보스 캐릭터를 플레이어 의상 슬롯으로 재배치하고, 색은 게임 보스 재질을 그대로
참조한다. 제나라의 머리·머리카락은 이 메시에 이미 들어있어서, 바닐라 머리·머리카락은 빈
껍데기로 덮어 겹침을 막는다.

## 저작권

배포물에는 Pocketpair 팰월드의 보스 메시 지오메트리가 포함된다(텍스처는 미포함 — 게임에서
참조). 다른 스킨 모드와 동일하게 **무료·비상업·출처표기** 선에서 공유한다.
